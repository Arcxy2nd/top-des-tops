      ❓ Comment utiliser le site

      Tableau de bord (Dashboard)

      C'est la page d'accueil. Elle affiche un graphique qui résume les scores de chaque joueur.

      Filtres : en haut, tu peux cliquer sur les noms de joueurs ou de Tops pour n'afficher que ceux-là. Tu peux aussi choisir une période (7 jours, ce mois, etc.).

      Types de graphique : les boutons Empilé, Groupé, Courbes, Radar, Donut, Classement changent la façon de voir les scores. « Classement » montre le total général de chaque joueur.

      Options : sous les boutons de type, des options apparaissent selon le graphique choisi (tri, échelle, totaux…).

      Export : les boutons en haut à droite (Infographie, CSV, Excel) permettent de télécharger le graphique ou les données.

      Saisir un Lot

      C'est ici qu'on enregistre des points. Chaque ligne du lot correspond à une entrée : un joueur, un Top, un nombre de points, et une date.

      Raccourcis de points : les boutons 1, 3, 5, 7… permettent de choisir rapidement. Tu peux aussi taper un nombre personnalisé.

      Plage de dates : coche « Plage » pour étaler une entrée sur plusieurs jours. Tu peux choisir « Répéter » (même score chaque jour) ou « Répartir » (le total est divisé sur les jours). Le sélecteur de jours de la semaine permet de ne garder que certains jours (ex : jours ouvrés).

      Description : champ optionnel en bas de chaque ligne pour ajouter un contexte.

      Dupliquer : le bouton 📋 copie une ligne existante pour aller plus vite.

      Paramètres

      On y gère les joueurs (nom, avatar, couleur) et les Tops (nom, description, emoji, couleur).

      Le barème en bas permet de définir des règles de points pour chaque Top (ex : « 🍺 Apéro → Ramener des chips = 3 pts »). C'est un aide-mémoire consultable via le bouton ⚖️ en bas à gauche.

      Notes

      Espace pour écrire des remarques libres sur chaque joueur.

      La barre rapide en haut permet d'ajouter une note en une seconde : choisis le joueur, tape la note, appuie sur Entrée.

      Les notes de chaque joueur sont visibles en dessous dans des blocs dédiés.

      Historique

      Toutes les entrées enregistrées, de la plus récente à la plus ancienne. Filtre par joueur ou par Top.

      Sélection : le bouton « ☑ Sélectionner » active un mode où tu peux cocher des entrées pour les supprimer en bloc ou leur donner la même description.

      Regroupement : les entrées identiques consécutives (même joueur, Top, points, description) sont regroupées. Clique sur le groupe pour voir chaque ligne individuelle.

      Chaque entrée peut être éditée (description) ou supprimée individuellement.

      Outils

      Rapport de santé : montre le nombre total d'entrées, celles à 0 point, et les orphelines (joueur ou Top supprimé).

      Nettoyage : deux boutons pour supprimer les entrées à 0 point ou les orphelines.

      Détection de lots répartis : scanne l'historique pour trouver des séries d'entrées identiques étalées sur plusieurs jours (typiquement créées avec « Répartir »). Tu choisis lesquels supprimer, avec double confirmation car c'est irréversible.

      Barème (bouton ⚖️)

      Le bouton flottant en bas à gauche ouvre un panneau qui liste toutes les règles de points définies dans les Paramètres. C'est un aide-mémoire rapide consultable depuis n'importe quel onglet.

      Raccourci clavier : appuie sur ? pour ouvrir/fermer.
